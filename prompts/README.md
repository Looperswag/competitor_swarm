# Agent 系统提示词

本目录存储各个 Agent 的系统提示词，以独立的 Markdown 文件形式维护。

## 文件说明

| 文件 | Agent | 说明 |
|------|-------|------|
| `scout.md` | 侦察专家 | 产品基础信息收集与分析 |
| `experience.md` | 体验专家 | 用户体验评估与 UI/UX 分析 |
| `technical.md` | 技术分析专家 | 技术栈推断与架构分析 |
| `market.md` | 市场分析专家 | 市场定位与竞争分析 |
| `red_team.md` | 红队专家 | 批判性分析与问题发现 |
| `blue_team.md` | 蓝队专家 | 优势识别与价值发现 |
| `elite.md` | 综合分析专家 | 跨维度综合分析与建议生成 |

## 修改提示词

1. 直接编辑对应的 `.md` 文件
2. 修改后需要重启程序使更改生效
3. 提示词内容会被程序启动时自动加载

## 提示词格式

提示词文件应包含完整的 Agent 系统提示词内容，可以使用 Markdown 格式增强可读性。程序会自动去除 YAML frontmatter（如果存在）。
